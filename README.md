# CR2ToJPG - Bulk CR2 to JPG Converter

A fast, free, and secure online tool for converting multiple Canon CR2 raw image files to JPG format instantly in your browser. No upload required - all processing happens locally on your device.

## 🌐 Online Usage

**🚀 [Try it online now](https://huashengjieguo.github.io/bulk-cr2-to-jpg/)** - No installation required!

## 📦 Self-Hosting

This is an **open source project** that you can:
- **Deploy yourself** on any web server
- **Customize** the code to fit your needs
- **Contribute** to the project development
- **Fork** and create your own version

- **GitHub Pages**: Fork and enable Pages
- **Netlify**: Drag and drop the files
- **Vercel**: Connect your GitHub repository
- **Any Web Server**: Upload files to your hosting provider
